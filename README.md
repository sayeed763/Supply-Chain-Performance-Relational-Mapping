Global Supply Chain & Logistics Optimizer
**Files:** `analyze_logistics.py`, `warehouse_performance.csv`, `shipments.csv`
* **Python:** Processed multiple CSVs to calculate "Days to Ship" metrics by handling complex date-time objects.
* **SQL:** Utilized date functions and aggregations (SUM/AVG) to identify underperforming warehouse locations.
* **Excel:** Built a "Logistics Scorecard" with line graphs to track shipping delays and used Conditional Formatting to flag late shipments.
